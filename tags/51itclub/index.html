    Hexo模块化主题【51itclub】

    发表于 2015-06-13

    主题简介

    本主题 51itclub主题 Based On(基于) ChenallV2.2主题修改而成
    所有ChenallV2.2有的特性本主题都有，包括子分类、iLink文章内链、ijs文章内嵌脚本等
    所有ChenallV2.2没有的特性也可以通过比对chenall.net和gitcafe.51itclub.net一览无遗

    Hexo在线帮助文档

    关于ChenallV2.2

    它是基于Hexo默认主题 light 修改而成的,采用 bootstrap 框架
    在尽量保留原有主题功能的同时采用模块化设计支持实现组件模块化.完全自定义,通过widgets你可以添加许多实用功能.可以自定义组合各种组件,像评论系统,统计系统等.
    大部份的JS/CSS等静态文件都使用CDN加载,使用了 bootcss 和 qiniu 的CDN服务.

    更新内容

    因为改动比较多，无法一一陈列，所以只挑几个想到的列出来
    1.添加返回顶部功能
    2.添加友情链接插件
    3.添加RSS订阅插件
    4.添加类jacman主题的页尾可旋转头像
    5.可悬浮导航栏添加Flash时钟和MP3播放器，播放曲目可在“Tools—>歌单刷新”菜单中查看
    6.像jacman主题一样，引用了三种第三方网页字体库’FontAwesome’、’fontdiao’和’covered_by_your_graceregular’，包括QQ、Github之类的图标大多出自’FontAwesome’，还有一些国内比较有名的网站，如豆瓣网站图标，则出自’fontdiao’字体库，’covered_by_your_graceregular’字体库一般用于英语，样式如页尾那段英文简介所示，颇有艺术感

    主题说明

    加上以下代码可以隐藏主题中的侧边栏，去掉即显示，因为默认是显示的，一般情况下代码加在文章id项之下

    widgets:
  theme: false

var str = '';
if (site.posts.length){
  str = '<ul class="list-group">';
  site.posts.sort('date', -1).limit(5).each(function(post){
    str += '<li class="list-group-item"><a href="' + config.root + post.path + '">' + post.title + '</a></li>';

    iLink 文章内链

    • tag 链接到标签 

       {% iLink tag:GRUB4DOS %}

      例:

      GRUB4DOS

    • cat 或 category 链接到分类
      例子: 

       {% iLink cat:原创 %}

      例:

      原创

    • 其它值会根据提供的信息自动链接到指定文章
      例子:

      1. 链接到源文件名 

        {% iLink source:_posts/grub4dos_dev.md %}

        例:

        在WINDOWS系统中搭建GRUB4DOS编译环境

      2. 链接到文章标题

        例:

        在WINDOWS系统中搭建GRUB4DOS编译环境

      3. 若某个文章页有额外添加了一些信息比如id: 804 

        {% iLink id: 804 %}

        例:

        在WINDOWS系统中搭建GRUB4DOS编译环境

    附加自定义内容

    如果你需要一些添加一些额外和主题无关的内容.这时就可以使用这个功能.
    在_modules目录中(主题或source都可以)新建一个文件夹partial.

    然后就可以通过partial里面的特定文件添加额外内容.

    具体文件列表:

    • body_start.ejs 标签<body>后.
    • body_end.ejs 标签</body>前.
    • head_start.ejs 标签<head>后
    • head_end.ejs 标签</head>前

    例子: 想要在head中添加一行meta信息就可以在head_start.ejs中添加.

    主题配置说明

    注: 配置中如果需要访问到本地路径,除非特别指定否则应该用css/theme.css不要写成/css/theme.css,前者是相对路径会自动添加config.root路径.后者是绝对路径.

    本主题特色: 把这个配置文件复制到source目录下并改名为_51itclub.yml则会优先使用该配置,这样可以避免由于升级主题或其它原因导致的配置丢失.更方便使用.
